# Платформа онлайн-репетиторства Online Tutors

Online Tutors — это современная веб-платформа, которая объединяет репетиторов и учащихся для проведения индивидуальных онлайн-занятий по различным учебным дисциплинам. Проект направлен на создание удобного, прозрачного и эффективного пространства для дистанционного обучения.

Решаемые проблемы
- Для учеников: сложность поиска проверенного репетитора с подходящей специализацией, гибким расписанием и прозрачным ценообразованием.
- Для репетиторов: отсутствие удобного инструмента для управления расписанием, проведения занятий, автоматизации оплат и привлечения новых учеников.
- Для образовательного процесса: необходимость в структурированном подходе к организации уроков, отслеживании прогресса и фиксации всех этапов взаимодействия.

## Инструкции по установке

Требования:
- Node.js 18+
- PostgreSQL 15
- Docker

Установка:
1. Клонируй репозиторий:
```bash
git clone <your-repo-url>
cd online_tutors_db
```
2. Установи зависимости:
```bash
npm install
```
3. Создай .env файл (опционально):
```env
DB_HOST=localhost
DB_USER=online_tutors_db
DB_PASSWORD=1111
DB_NAME=online_tutors_db
DB_PORT=5434
APP_PORT=8080

```
4. Запуск через Docker 
```bash
docker compose up --build
```

После запуска:
- API: http://localhost:8080
- Swagger: http://localhost:8080/api-docs

## Использование

Пример подключения:
```js
const axios = require('axios');

axios.get('http://localhost:8080/api/users')
  .then(res => console.log(res.data));
```
Основные эндпоинты:
- Получить пользователей:
  ```GET /api/users```
- Получить пользователей по роли:
  ```GET /api/users/raw/by-role?role=student```
- Получить все предметы:
  ```GET /api/subjects```
- Поиск предметов по названию:
  ```GET /api/subjects/raw/search?name=Math```
- Получить все связи преподаватель-предмет:
  ```GET /api/tutor-subjects```
- Фильтрация связей преподаватель-предмет по цене:
  ```GET /api/tutor-subjects/raw/by-price?minPrice=10&maxPrice=50```
- Получить все уроки:
  ```GET /api/lessons```
- Получить уроки по диапазону дат:
  ```GET /api/lessons/by-date?start=2026-04-01T00:00:00.000Z&end=2026-04-30T23:59:59.000Z```
- Получить все отзывы:
  ```GET /api/reviews```
- Получить статистику отзывов по преподавателю:
  ```GET /api/reviews/raw/tutor-stats?tutorId=2```

Swagger
Документация API доступна по адресу: /api-docs
